# MassMotion

> **Emotion-first short drama discovery in 60 seconds.**

Find your next binge-worthy Chinese short drama by mood. No signup required.

## 🎯 Core Features

- **Emotion-driven discovery**: Select your mood (😭 Cry, 💕 Sweet, 🔥 Hype, 😂 Funny, 🩸 Dog Blood)
- **60-second flow**: From landing to watching in under a minute
- **Quality curation**: Every drama rated by scoreV0 (0-10) with one-line recommendation
- **HE/BE tags**: Know the ending type before you commit (Happy Ending / Bad Ending)
- **Free YouTube content**: All dramas are free to watch on YouTube

## 🚀 Tech Stack

- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Deployment**: Vercel

## 📊 MVP Features

✅ Mood chip selection
✅ "Get me something good" recommendation engine
✅ Drama cards with HE/BE badges
✅ Vibe icons and trope tags
✅ YouTube playback integration
⏳ Anonymous reactions (🔥😭💕😂) with Supabase
⏳ Watch Later list (localStorage)
⏳ 30-40 curated dramas

## 📝 Data Structure

Every drama has: ending (HE/BE), vibes, tropes, scoreV0 (0-10), and one-line "why"

## 🎨 Design Philosophy

**Every element has meaning. Nothing is decoration.**

Simplified from marsdrama. Emotion-first, not diary-first.
